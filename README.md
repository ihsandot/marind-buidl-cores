# ⚡ Marind Buidl Cores

Automated, real-time blockchain telemetry pipelines designed to aggregate cluster health metrics, execution payload data, and query latency across top-tier Web3 protocols.

## 🛰️ Architecture Overview

These standalone TypeScript modules act as the processing engine for the Marind Collective ecosystem. By executing direct JSON-RPC handshakes without relying on central database proxies, the pipelines guarantee trustless data integrity directly from decentralized validator node networks.

## 📁 Repository Structure

* `pipelines/solana-telemetry.ts` — Fetches real-time Block Height and current Epoch data straight from the Solana Mainnet cluster endpoint.
* `pipelines/hyperliquid-core.ts` — Connects to the Hyperliquid L1 API info network to extract cumulative 24h trading volume parameters.

## 📡 Live Integration Details

The telemetry outputs generated by these open-source modules are ingested dynamically by the interface terminal dashboard layer at `marind-collective`.
